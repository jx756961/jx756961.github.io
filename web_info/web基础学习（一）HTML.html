HTML

  • Hyper Text Markup Language 超文本标记语言

  • HTML发展历史了解,目前网页常用HTML5。不要在意图片上的问题！

  • 所有HTML都遵循W3C标准。即：万维网联盟行业标准的制定者

  • HTML页面主结构

  • 本部分知识点实战demo.html

	<!--网页标题-->

</head>
