web基础学习（十四）CSS3多列布局

上一节：CSS3弹性盒模型布局——————下一节：CSS3响应式布局

多列（ Multi-column ）是一个CSS3新增布局模块，官方称为Multiple column layout，可以比较轻松的实现多列布局。

  1. column-width

    column-width：<length> | auto
    设置列的宽度

  2. column-count

    column-count：<integer>
    设置列的个数

  3. columns

    columns：[ column-width ] || [ column-count ]
    复合属性，设置列的宽度和个数

  4. column-gap

    column-gap：<length> | normal
    设置列与列之间的间隙

  5. column-rule

    column-rule：[ column-rule-width ] || [ column-rule-style ] || [ column-rule-color ]
    设置列与列之间的边框, 类border。

  6. column-fill

    column-fill：auto | balance
    设置所有列的高度是否统一

  7. column-span

    column-span：none | all
    对象元素是否横跨所有列

  • 案例展示：
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
    <style type="text/css">
        body {
            margin: 0;
        }

        header {
            height: 100px;
            width: 100%;
            display: block;
            background-color: #fff;
            box-shadow: 0 3px 2px rgba(0, 0, 0, 0.4);
            position: fixed;
            top: 0;
            left: 0;
            text-align: center;
            line-height: 100px;
        }

        header h1 {
            margin: 0;
        }

        .container {
            padding-top: 110px;
            /*设置列的宽度 多个浏览器兼容*/
            -webkit-column-width: 300px;
            -moz-column-width: 300px;
            -o-column-width: 300px;
            column-width: 300px;
            /*设置列的个数*/
            -webkit-column-count: 3;
            -moz-column-count: 3;
            -o-column-count: 3;
            column-count: 3;
            /* columns: 300px 3; */
            /*设置列与列之间的间隙*/
            -moz-column-gap: 50px;
            /*设置列与列之间的边框*/
            -webkit-column-rule: 2px dashed #333;
            -moz-column-rule: 2px dashed #333;
            -o-column-rule: 2px dashed #333;
            column-rule: 2px dashed #333;
        }

        .title {
            text-align: center;
            /*对象元素横跨所有列*/
            -webkit-column-span: all;
            -moz-column-span: all;
            -o-column-span: all;
            column-span: all;
        }
    </style>
